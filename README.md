# Ghost Tracker
IP Intelligence Platform for red team reconnaissance.

## Usage
```bash
python3 ghost_tracker.py 8.8.8.8
python3 ghost_tracker.py --me
python3 ghost_tracker.py 1.1.1.1 --save
python3 ghost_tracker.py --batch targets.txt
```

## Features
- Geolocation, ASN, hostname resolution
- Proxy / VPN / Tor / datacenter detection
- Risk scoring (0-100)
- JSON report export
- Batch mode

## Ethical use
Passive OSINT only. Use on IPs you own or have authorisation to investigate.
